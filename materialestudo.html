O que é HTML?

HTML é a linguagem responsável por estruturar páginas da web. Com ela organizamos textos, imagens, links e tabelas.

Principais Tags:

  • <html>
  • <head>
  • <body>
  • <p>
  • <a>
  • <img>

O que é CSS?

CSS é a linguagem usada para estilizar o HTML. Podemos alterar cores, tamanhos e posicionamento.

Propriedade Função
color Cor do texto
background-color Cor de fundo
font-size Tamanho da fonte
